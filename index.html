Мультипоиск по артикулам

Одновременный поиск по МойСклад (наличие), удалённому складу и японскому поставщику (под заказ). Сравнение цен и наличия в реальном времени.

🤖

AI-обработка запросов

GPT-модель фильтрует спам, извлекает ФИО/телефон/адрес из свободного текста, переводит японские названия деталей и определяет релевантность сообщений.

📦

Автозаказ в МойСклад

Полный цикл оформления: поиск → корзина → данные клиента → создание заказа и контрагента в МойСклад API. Без участия менеджера.

🇯🇵

Импорт из Японии

Интеграция с японским поставщиком: запрос цен в йенах, автоматический пересчёт в рубли с учётом доставки и наценки. Массовая оценка списка деталей.

💬

Telegram + Avito

Единый бот работает в Telegram (aiogram 3) и Avito (webhook). Общая логика обработки, единый склад и база клиентов для обоих каналов.

⚙️

Машина состояний

Детерминированная FSM для оформления заказа: надёжная без ИИ-галлюцинаций. Автосброс при смене намерения, распознавание телефонов, адресов и ФИО.
